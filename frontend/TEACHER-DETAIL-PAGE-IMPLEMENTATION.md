## 📋 **المميزات / Features**

### 1. **نظرة عامة شاملة / Comprehensive Overview**
- بطاقات إحصائية في الأعلى:
  - الرتبة العلمية
  - سنوات الخبرة
  - عدد المواد المكلف بها
  - ساعات التدريس الأسبوعية

### 2. **المعلومات الشخصية / Personal Information**
- الاسم الكامل (بالعربية والإنجليزية)
- البريد الإلكتروني
- رقم الهاتف
- عرض الأيقونات بجانب كل معلومة

### 3. **المعلومات الأكاديمية / Academic Information**
- المؤهل العلمي (دكتوراه، ماجستير، إلخ)
- الرتبة العلمية (أستاذ، أستاذ مشارك، إلخ)
- سنوات الخبرة
- **الأقسام**: عرض جميع الأقسام مع تمييز القسم الرئيسي
- **التخصصات الدقيقة**: عرض جميع التخصصات كبطاقات

### 4. **المعلومات المالية / Financial Information**
- الراتب الأساسي
- معدل الساعة الإضافية
- ساعات التدريس الأسبوعية المطلوبة

### 5. **المواد المكلف بها / Assigned Subjects**
- قائمة كاملة بالمواد
- عرض تفاصيل كل مادة:
  - اسم المادة والكود
  - القسم
  - السنة الأكاديمية
  - الفصل الدراسي
  - المدرس الرئيسي (Primary Teacher)
  - صلاحيات (تعديل الدرجات، أخذ الحضور)
  - ملاحظات
- **إدارة المواد**:
  - إضافة مادة جديدة
  - تعديل التكليف
  - حذف التكليف

### 6. **الجدول الأسبوعي والتوفر / Weekly Availability**
- جدول تفاعلي يعرض توفر المدرس
- 5 أيام عمل (الأحد - الخميس)
- 5 فترات زمنية يومياً (8:00 صباحاً - 6:00 مساءً)
- رموز مرئية:
  - ✅ (أخضر) = متاح
  - ❌ (رمادي) = غير متاح
- مفتاح توضيحي للألوان

### 7. **الإجراءات / Actions**
- زر "تعديل البيانات" في الأعلى
- زر "العودة للقائمة"
- إضافة/تعديل/حذف المواد

## 🚀 **كيفية الاستخدام / How to Use**

### 1. **الوصول للصفحة / Access the Page:**
- من صفحة قائمة المدرسين
- اضغط على أيقونة العين 👁️ بجانب أي مدرس
- سيتم الانتقال لصفحة التفاصيل الكاملة

### 2. **عرض المعلومات / View Information:**
- جميع المعلومات معروضة مباشرة
- لا حاجة للتنقل بين الصفحات
- التمرير لأسفل لرؤية المزيد

### 3. **إدارة المواد / Manage Subjects:**
- اضغط "إضافة مادة" لتكليف جديد
- اضغط أيقونة القلم ✏️ للتعديل
- اضغط أيقونة السلة 🗑️ للحذف

### 4. **تعديل البيانات / Edit Data:**
- اضغط "تعديل البيانات" في الأعلى
- سيتم الانتقال لصفحة التعديل الكاملة
- يمكن تعديل جميع المعلومات بما فيها الجدول الأسبوعي
