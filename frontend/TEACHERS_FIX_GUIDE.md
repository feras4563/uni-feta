# دليل إصلاح صفحة المدرسين

## المشكلة
الملف `src/pages/Teachers.tsx` يحتوي على نصوص عربية تظهر كعلامات استفهام بسبب مشكلة في الترميز.

## الحل
يجب استبدال النصوص التالية:

### أيام الأسبوع
```
تعديل → الأحد
تعديل?? → الاثنين  
تعديلحذف → الثلاثاء
تعديلحذف → الأربعاء
تعديل? → الخميس
```

### العناوين والأزرار
```
?رقم الهاتف? → إدارة المدرسين
تعديل حذف نشط تعديل → إضافة مدرس جديد
قائمة المدرسين → تصدير البيانات
```

### الإحصائيات
```
تعديل? ?رقم الهاتف? → إجمالي المدرسين
تعديل نشط إضافة مدرس جديدحذف → المدرسون النشطون
تعديل? → الأساتذة
مدرسون جدد? (تعديل) → متوسط الخبرة (سنوات)
```

### عناوين الجدول
```
تعديل → الاسم
?مدرسون جدد?? → المؤهل العلمي
تعديل? → الأقسام
تعديل? → الخبرة
?قائمة المدرسين?? → البريد الإلكتروني
تعديلنشط → الإجراءات
```

### المؤهلات
```
تعديل → أستاذ
مدرسون جدد → أستاذ مشارك
مدرسون جدد → أستاذ مساعد
تعديل → محاضر
```

### المستويات التعليمية
```
تعديل?? → دكتوراه
تعديل?? → ماجستير
تعديلنشط → بكالوريوس
تعديل نشط → دبلوم عالي
تعديل → دبلوم
```

### النصوص العامة
```
حذف نشط → غير محدد
نشط → محدد / آخر
تعديل? → السابق
تعديل? → التالي
حذف → عرض / حذف
تعديل → تعديل / رئيسي
```

### رسائل التنبيه
```
تم تحديث بيانات المدرس بنجاح
تم إضافة المدرس بنجاح
تم حذف المدرس بنجاح
تم حذف التعيين بنجاح
حدث خطأ أثناء حفظ البيانات
هل أنت متأكد من الحذف؟
```

## ملاحظة مهمة
الملف الحالي تضرر بسبب محاولات التعديل السابقة. يُنصح بإعادة إنشائه من نسخة احتياطية أو من Git.

## الخطوات المقترحة
1. استعادة الملف من Git: `git checkout origin/main -- src/pages/Teachers.tsx`
2. تطبيق التعديلات يدوياً أو باستخدام أداة البحث والاستبدال
3. التأكد من عدم وجود أخطاء في الترميز

## ملف النسخة الاحتياطية
تم حفظ نسخة من الملف التالف في: `src/pages/Teachers.tsx.broken`
